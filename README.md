# GO-TODO-CLI APP
Go Todo-CLI App with gui

### Features
- [x] Add Task
- [x] Complete Task
- [ ] Add tags to a task
- [ ] Task Tracker

### RUN
```
go run cmd/todo-cli/main.go
```
